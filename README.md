This tool provide a feature to fetch the data from an ECommerce website link, extract the Title and Description of a product, Insert into MySql database. When a user search as a query, it will provide the product details.
Tech Stack
Python : BeautifulSoup, SentenceTransformer
PHP : RestAPI in Laravel
Database : MySql
AI : groq

Python library will fetch the data from given URL, extract required data
Rest API will store the data and fetch details to the given query
Groq AI to answer the user query
